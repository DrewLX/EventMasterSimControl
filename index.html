Event Master Simulator Controller

About

This app is an evolution upon the work started by Nick Lloyd. I had the pleasure of knowing Nick since we were both at school together, and continued to regularly work together until he died in October 2018.

His E2 Simulator XML website was an invaluable resource and tool, however I wanted to take it a step further, and make the simulator even easier to use.
So now you can simply configure and hit start. The app generates the XML file and starts up the simulator all for you in one window.

Installation

Mac

Download v0.7
Simply open the zip, and drag the app into your applications folder.

Windows

Download v0.7
Download the .zip file and extract it. Simply run e2-sim-control.exe
I'm working on an installer and certificates to make this better but I'm not a windows user. If someone can help, please get in touch!

View all the versions here: GitHub Releases

Setup

Once the app is running, the first time you'll need to click the 'Set Path' button. This should be the root of the Event Master version you wish to use.

For example on a mac something like /Applications/EventMaster-6.2.47841

On windows it would be C:\Barco\EventMaster-6.2.47841

Simply configure your Event Master, select your cards and hit 'Start'.

History

Without this turning into an obituary, I do want to recognise the help, support, time and love that Nick gave to our community. I felt I should continue his work and create this app. If Barco create something better I will certainly welcome it, but in the meantime I hope this can be of use to the many Event Master operators and technicians around the world.
I have used as much of Nick's work as possible. All his styling, html and javascript still remains. The rest of this app just adds the new functionality of passing the xml directly to the drive and executing the simulator.
